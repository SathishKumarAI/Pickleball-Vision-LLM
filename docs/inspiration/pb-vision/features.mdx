# PB Vision — Feature Set

> Captured from public marketing + help-center content for inspiration.

## Shot speed & shot stats
- Average **serve** and **drive** speeds, benchmarked vs other players.
- **Serve speed-to-depth** and the same for **returns**.
- Analyzes every **3rd shot** and classifies it: **drop · drive · hybrid · lob**.
- Net/out errors: how often you hit **into the net** or **out of bounds**.

## Kitchen & court movement ("Court Coverage")
- How much a player **moves** during play.
- **Positioning** with different partners.
- **Efficiency reaching the kitchen line** when serving.
- **Court-coverage heatmaps**.

## Rally analytics
- Pulls out **attacks (poaches)**, **hand battles**, and **longest rallies**.

## Automatic highlights
- Auto-generated clips: long rallies, hand battles, **Ernes**, **ATPs**.

## Performance breakdown
- **Court coverage heatmaps**, **3D shot breakdowns**, detailed stats.
- Serve/return **depth** monitoring.

## Skill rating + ecosystem
- A composite **PB Vision skill rating** (see `skill-rating.mdx`).
- **Shot Explorer** tool; **data export**; **DUPR** rating submission integration.

## Workflow
1. **Record** on phone/computer → 2. **Upload** → 3. **Review** AI analysis/stats
→ 4. **Analyze** via dashboards → 5. **Share** clips → 6. **Export** data.

## Platforms
Web app + **iOS app** (App Store). Targets recreational → competitive players,
coaches, and clubs.
